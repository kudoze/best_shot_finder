# 📸 Best Shot Finder

Ein kleines KI-gestütztes Tool zur Bewertung von Fotos – lokal oder als Streamlit-App.  
Es berechnet Schärfe und Belichtung als Bewertungskriterium.

## 🔍 Features

- Analyse der Bildschärfe (Laplacian-Methode)
- Helligkeitsbewertung (Durchschnitt aller Pixel)
- Streamlit-Frontend für einfache Bedienung

## 🧪 Beispiel

```bash
python analyzer.py --image pfad/zu/deinem_foto.jpg
